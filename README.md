  <h1>🎮 Número Secreto com Tela Interativa</h1>
  <p>Este projeto é uma versão aprimorada do clássico jogo do número secreto, agora com uma <strong>interface gráfica</strong> e suporte a <strong>comandos de voz</strong>!</p>

  <h2>🔗 Demonstração</h2>
  <p>Você pode jogar diretamente pelo navegador acessando: 
    <a href="https://numero-secreto-tela.vercel.app" target="_blank">numero-secreto-tela.vercel.app</a>
  </p>

  <h2>📁 Estrutura do Projeto</h2>
  <ul>
    <li><code>index.html</code> – Página principal do jogo</li>
    <li><code>style.css</code> – Estilos visuais</li>
    <li><code>app.js</code> – Lógica do jogo e integração com voz</li>
    <li><code>img/</code> – Imagens utilizadas na interface</li>
  </ul>

  <h2>🚀 Funcionalidades</h2>
  <ul>
    <li>Interface amigável e responsiva</li>
    <li>Reconhecimento de voz para entrada do jogador</li>
    <li>Feedback visual e sonoro</li>
    <li>Mensagens dinâmicas de acerto ou erro</li>
  </ul>

  <h2>🛠️ Tecnologias Utilizadas</h2>
  <ul>
    <li>HTML</li>
    <li>CSS</li>
    <li>JavaScript</li>
    <li>Web Speech API</li>
  </ul>
